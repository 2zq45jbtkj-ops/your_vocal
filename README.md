# Домашка по вокалу — Telegram Mini App

Мини-приложение с домашними заданиями для учеников. Статический сайт без сборки:
что лежит в репозитории, то и работает. Любой пуш в `main` автоматически деплоится на Vercel.

## Структура

```
index.html            — точка входа, подключает Telegram WebApp SDK
styles.css            — стили (фирменная палитра)
app.js                — логика: список уроков, экран урока, тесты, разметка дыхания
data/lessons.json     — список уроков
data/lesson-01.json   — контент урока 1 «Дыхание»
```

## Как наполнять уроки (код трогать не нужно)

### Добавить новый урок

1. Создай файл `data/lesson-02.json` по образцу `data/lesson-01.json`.
2. В `data/lessons.json` добавь запись в массив `lessons`:

```json
{ "id": 2, "title": "Название урока", "ready": true, "file": "data/lesson-02.json" }
```

Уроки без записи (или с `"ready": false`) показываются в списке со статусом «скоро».

### Разметка дыхания в текстах песен

В строках примера и вопросов теста используются пометки:

- `[В]` — полный вдох (отображается как ∨, терракотовый)
- `[д]` — добор (отображается как ’, сине-серый)

### Аудио распевок

Положи mp3 в папку `audio/` и укажи путь в поле `audio`:

```json
{ "topic": "Губной вибрант", "description": "…", "audio": "audio/vibrato.mp3" }
```

Пока `audio: null` — показывается заглушка «аудио появится позже».

- подложки: `#B8CCD9`, `#D6A289`

## Привязка к Telegram-боту

1. В Telegram открой [@BotFather](https://t.me/BotFather).
2. `/newbot` (если бота ещё нет) — задай имя и username.
3. `/newapp` — выбери бота, заполни название и описание, в поле **Web App URL**
   вставь адрес продакшна на Vercel, задай короткое имя приложения.
4. Готово: приложение открывается по ссылке `t.me/<бот>/<короткое_имя>`
   и через кнопку меню бота (настраивается в BotFather: Bot Settings → Menu Button).
